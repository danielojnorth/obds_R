3. Create input gene lists.

  • Create these 3 data frames by subsetting the original one:
    • all_genes_tested_df object containing table of all genes tested in the differential expression analysis, which will be the background set
    • upreg_genes_df object containing all significantly upregulated genes (‘padj’ < 0.05 & ‘log2FoldChange’ > 1)
    • downreg_genes_df object containing all significantly downregulated genes (‘padj’ < 0.05 & ‘log2FoldChange’ < -1)
all_genes_tested_df <- all_genes_df %>%  filter(!is.na(padj))

upreg_genes_df <- all_genes_df %>% filter(padj<0.05 & log2FoldChange > 1)

downreg_genes_df<- all_genes_df %>% filter(padj<0.05 & log2FoldChange < -1)

4. Obtain list of gene sets / pathways

  1. The fgsea() function requires (1) a list of gene sets / pathways to test enrichment of, and (2) a named vector of gene-level statistic (log2FoldChange), where the names should be the same as the gene names in the gene sets / pathways. First, let’s create our named vector of log2FoldChange values.
  • See ?tibble::deframe for help here. The deframe() function converts 2-column data frames to a named vector or list, using the first column as name and the second column as value.
ranks <- deframe(all_genes_tested_df_GSEA)
  • Sort the genes based on the ranking metric.
ranks <- sort(ranks, decreasing = TRUE)
plot (ranks)
  1. Obtain gene sets / pathways. Let’s use the Hallmark gene sets from MSigDB. “Hallmark gene sets summarise and represent specific well-defined biological states or processes and display coherent expression. These gene sets were generated by a computational methodology based on identifying overlaps between gene sets in other MSigDB collections and retaining genes that display coordinate expression.”

We can either download GMT files directly from MSigDB or use the misgidbr R package. We’ll do the latter. Check the documentation for the functions we need.

                  stats = ranks,

  • Convert result to tibble and arrange by normalised enrichment score
fgseaResTidy <- fgseaRes %>% as_tibble() %>%
